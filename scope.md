# Project Scope Statement

Project Name: Personal Introduction Webpage  
Completed by: Zerrie Mack  
Projected start: 01/19/26
Projected end: 01/30/26

## Project Purpose
Create a simple personal introduction webpage for the course that presents who I am, what I study, and where to find my GitHub.

## Project Description
Build one HTML page named index.html. The page will include a page title, my name as the main heading, a subheading with my major or interests, two short paragraphs about my academic and career goals, an unordered list of hobbies or interests, and a working link to my GitHub profile.

## Desired Results
1. A working index.html page that loads in a browser with all required elements visible.
2. Headings use h1 and h2 correctly.
3. Two paragraphs clearly describe my goals.
4. Unordered list contains at least three hobbies or interests.
5. GitHub profile link works and goes to the correct page.
6. Repository includes scope.md and index.html.

## Exclusions
1. No extra pages beyond index.html.
2. No advanced styling requirements beyond basic readable HTML.
3. No JavaScript requirements.
4. No need for images unless I choose to add them later.

## Constraints
1. Must follow the assignment requirements exactly.
2. Must be written in HTML and submitted through GitHub.
3. Must be packaged as a zip file for submission.
